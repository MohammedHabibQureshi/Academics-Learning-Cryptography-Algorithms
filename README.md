# 🔐 Cryptography Algorithms in Python

## Overview

This repository contains implementations of classical and modern cryptographic algorithms developed as part of academic coursework and cybersecurity learning.

The project demonstrates fundamental encryption techniques, secure key exchange protocols, stream and block ciphers, public-key cryptography, and cryptographically secure random number generation through practical Python implementations.

The primary objective is to strengthen the understanding of cryptographic concepts by implementing algorithms from scratch and analyzing their behavior.

---

## Key Features

* Implementation of 17 cryptographic algorithms
* Classical and modern encryption techniques
* Public-key and symmetric cryptography
* Secure key exchange mechanisms
* Cryptographic random number generators
* Educational Python implementations
* Structured repository organization

## Algorithms Implemented

### Classical Cryptography

| Algorithm                     | Type                        |
| ----------------------------- | --------------------------- |
| Caesar Cipher                 | Substitution Cipher         |
| Affine Cipher                 | Monoalphabetic Substitution |
| Columnar Transposition Cipher | Transposition Cipher        |
| Hill Cipher                   | Polygraphic Cipher          |
| One-Time Pad                  | Perfect Secrecy Cipher      |
| Playfair Cipher               | Digraph Substitution Cipher |
| Rail Fence Cipher             | Transposition Cipher        |
| Vigenère Cipher               | Polyalphabetic Cipher       |

---

### Modern Cryptography

| Algorithm                     | Category                |
| ----------------------------- | ----------------------- |
| AES Key Generation            | Symmetric Cryptography  |
| Diffie-Hellman Key Exchange   | Key Exchange Protocol   |
| ElGamal Cryptosystem          | Public Key Cryptography |
| RC4 Encryption                | Stream Cipher           |
| RSA Algorithm                 | Public Key Cryptography |
| Simplified DES Encryption     | Block Cipher            |
| Simplified DES Key Generation | Key Scheduling          |

### Random Number Generators

| Algorithm                           | Category                      |
| ----------------------------------- | ----------------------------- |

## Running the Programs

Clone the repository:

```bash
git clone https://github.com/MohammedHabibQureshi/Academics-Learning-Cryptography-Algorithms.git
```

Navigate to the repository:

```bash
cd Academics-Learning-Cryptography-Algorithms
```

Run any algorithm:

```bash
python Caesar-Cipher.py
```
